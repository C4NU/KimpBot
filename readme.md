# Kimchi Premium Telegram Bot

## Description
Get 5 coins' (XRP, XLM, TRX, EOS, DOT) kimchi premium, and send to Telegram Bot message.

## Requirements

1. Python 3.9
2. Upbit API
   1. Get Upbit API  from UPBIT OPEN API
3. Binance API
4. Telegram Bot
   1. BotFather
5. PythonAnywhere

## Install

### Install requirements.txt

```python
pip install -r requirements.txt
```
